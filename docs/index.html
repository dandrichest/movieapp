Movie App Assignment

This repository contains an ASP.NET MVC application for the Movie App assignment.

The GitHub Pages site is showing this static landing page because GitHub Pages cannot host the server-side MVC application directly.

To run the application, clone the repository and start it locally using .NET 10:

dotnet run --project MvcMovie\MvcMovie.csproj

Repository: github.com/dandrichest/movieapp

Note: The actual app is an ASP.NET MVC site, not a static Pages site.
